# **App Name**: QuizWhiz

## Core Features:

- Quiz Start: Start a new quiz from a selected topic.
- Multiple Choice Questions: Display multiple-choice questions with selectable answers.
- Immediate Feedback: Provide instant feedback after answering each question.
- Final Score Summary: Display the final score and a summary of correct/incorrect answers.

## Style Guidelines:

- Primary color: Bright green (#B0FF47) for an energetic and positive learning environment.
- Background color: Soft green (#E8FFD1), slightly desaturated version of the primary color to keep the interface light.
- Accent color: Yellow (#FFE54C), an analogous hue to the primary green, chosen to make important elements stand out.
- Font pairing: 'Space Grotesk' (sans-serif) for headings, and 'Inter' (sans-serif) for body text, providing a clean, tech-forward aesthetic that remains highly readable.
- Use clear and simple icons related to educational topics for better user understanding.
- Clean and intuitive layout for easy navigation and focused learning.
- Subtle animations and transitions to enhance user engagement.
